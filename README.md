# 📝 To-Do List

Basit ve işlevsel bir **to-do list** uygulaması. Kullanıcılar görev ekleyebilir, tamamlayabilir ve görev durumlarını takip edebilir.

---

## 🚀 Özellikler
- Görev ekleme
- Görevleri tamamlama / tamamlanmamış olarak işaretleme
- Tamamlanan ve tamamlanmayan görevlerin sayısını gösterme
- Temiz ve anlaşılır arayüz

---

## 📂 Proje de kullanılanlar:
- HTML
- CSS
- JavaScrpit

---

## 🖼️ Arayüz
• 	Başlık: "to-do list"
• 	Input kutusu: Yeni görev eklemek için
• 	Buton: Görev ekleme
• 	Liste: Eklenen görevlerin görüntülendiği alan
• 	Sayaçlar: Tamamlanan ve tamamlanmayan görevlerin sayısı
